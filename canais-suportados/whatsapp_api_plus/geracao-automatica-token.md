# Geração automática TOKEN

Com essa configuração ao cadastrar novo canal sistema vai gerar token automático no seu servidor uazapi e cadastrar automaticamente no whazing tornando conexão mais fácil e transparente. Muito simples so prescisa url servidor e token ADMIN. Essa configuração somente valida novos canais, os que já estiverem cadastrados no sistema não serão alterados. Caso houver erro gerar token canal não sera cadastrado ou atualizado.
