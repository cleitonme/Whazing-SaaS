# Gerenciamento de Tokens Plus

Objetivo para quem adquiri tokens avulsos ou pacotes, gerenciar mesmos pelos sistemas atribuindo automaticamente aos canais ao fazer migração entre apis ou cadastro novos canais de forma transparente e facilitando atendimento do dia a dia. Para cliente cadastrar canal de forma totalmente transparente sem pedir dados adicionais facilitando.

Está opção esta disponivel no painel SaaS na aba canais

Super simples Adiciona um canal preenche token e seleciona servidor

<figure><img src="../../.gitbook/assets/image.png" alt=""><figcaption></figcaption></figure>

Ao cadastra novo canal ou fazer migração canal sistema vai buscar esses tokens pre cadastrados e usar mesmo, caso não encontre nenhum disponivel ira da erro, Ao apagar canal ou migrar outra api canal ficara disponivel novamente para outro cliente usar.

Sera possivel buscar por canal e verificar o que ta em uso ou disponivel.
